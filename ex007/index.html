Negrito / Destaque

Italico

nesta frase vamos usar um termo em italico ultizando a TAG < i >

nessa frase usaremos a TAG < Em > para deixar em italico porem essa tag deixa o codico mais semantico

Aula de citacoes , codigo fontes

Quando eu quiser usar um codigo de alguma linguaguem de programacao eu uso a TAG < CODE > para deixar as letras do mesmo tamanho(espacamento). Assim fica mais facil de identificar um codigo exemplo:


a = 3 b = 4 soma = a + b print('A soma de a + b é: soma')
porem o codigo nao fica identado como deveria ser para isso usamos a TAG <PRE > para ficar Assim
